# Express API Demo

I created a simple api example using [Expressjs](http://www.expressjs.com)

## Installation

- clone the repository
```bash
git clone https://github.com/ehsanghaffarii/express-api-demo.git
```

- install dependencies
```bash
npm install
```

- start the application
```bash
npm start
```
